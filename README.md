# JWPUB

Download custom jwpub files in different languages for JW Library

Available languages:
1. Spanish
2. English
3. Yoremnokki
4. Hiligaynon
5. Brazilian Portuguese
6. German

Installation:
1. Open the file using JW Library.
2. Go to Library
3. Then to the appropriate category

____
____
